Sie können ForAnyList für alle Arten von Listen verwenden, z.B. für Einkaufslisten, Checklisten oder Aufgabenlisten.

Tragen Sie einfach alles, was Sie nicht vergessen möchten, in ForAnyList ein. Das können Tausende von Notizen, Fotos, Videos, Sprachaufnahmen oder Dokumenten sein. Dank der praktischen Ordnerstruktur und der Suchfunktion finden Sie Ihre Notizen schnell wieder.

Mit ForAnyList können Sie Ihre Listen auf intuitive Weise verwalten. Sie können - im Falle einer Aufgabenliste - eine Aufgabe einfach an eine beliebige Position ziehen.

So haben Sie die vollständige Kontrolle über die Reihenfolge, in der die Aufgaben in Ihrer Aufgabenliste angezeigt werden (z.B. dringende Aufgaben an der Spitze).

Auch können Sie die Farbe einer Aufgabe ändern, um z.B. den Status einer Aufgabe anzuzeigen, oder einfach nur, um eine bestimmte Aufgabe in der Aufgabenliste zu markieren.

Um Ihre Listen kompakt und übersichtlich zu halten, können Sie Ordner (Unterlisten) erstellen. Ziehen Sie eine Aufgabe einfach auf einen Ordner. Ein Ordner kann auch weitere Ordner enthalten, und diese dann weitere Ordner. Die Anzahl der Ebenen ist unbegrenzt.

Einfach und sicher bestellen

Optional können Sie eine Aufgabe als dringend markieren, oder Sie verknüpfen eine Aufgabe mit einer oder mehreren Personen und/oder Orten. Sie können Ihre Sammlung von Aufgaben auf vier verschiedene Arten (Abschnitte) beobachten:

1. Alle Notizen: Alle Ihre Listen mit Notizen; 2. Heute: Aufgaben, die kurzfristige Aufmerksamkeit benötigen; 3. Personen: Aufgaben die mit Personen verbunden sind. Also je eine Aufgabenliste pro Person; 4. Orte: Aufgaben die mit Orten verbunden sind. Also je eine Aufgabenliste pro Ort;

Eine weitere nützliche Funktion ist das Archiv. Damit können Sie gelöschte/abgeschlossenen Aufgaben anzeigen oder wiederherstellen. Zeigen Sie beispielsweise alle abgeschlossenen Aufgaben für „Project X“ im letzten Monat. Holen Sie „versehentlich“ gelöschte Notizen wieder aus dem Archiv. Oder ergänzen sie eine Einkaufsliste mit archivierten Produkte (die Sie zuvor gekauft haben), also ohne die Notwendigkeit, diese Produkte neu einzugeben.
